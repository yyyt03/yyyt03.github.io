关于实验室

本实验室依托于大学考古系建立，专注于土壤微形态学、实验考古学以及残留物分析。

我们的目标是通过跨学科的方法，将自然科学技术引入考古学研究，从微观层面揭示古代人类的生活方式、技术工艺以及环境适应策略。实验室配备了先进的偏光显微镜、扫描电镜以及实验考古操作台。

核心研究领域

结合传统考古学理论与现代科学技术，我们主要关注以下几个方向：

土壤微形态学

通过制作土壤薄片，在显微镜下观察沉积物的微观结构，重建遗址的形成过程及人类活动痕迹。

实验考古学

通过模拟古代技术流程（如石器打制、陶器烧制），验证考古假设，理解古代工匠的技术选择。

残留物分析

提取并分析陶器、石器表面的淀粉粒、植硅体及脂质残留，复原古代人类的饮食结构与资源利用。
